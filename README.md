## 🧠 Cách làm việc (Workflow)

- Mỗi người làm nhánh riêng
- Push lên nhánh cá nhân → tạo Pull Request vào `dev`, thêm người Review và người đó check rồi merge khi thấy ổn
- Chỉ merge `dev` → `main` khi bản ổn định
